# GIT

## Criar e mudar para um novo Branch

```
git checkout -b nomeDoBranch
```

## Merge com outro branch

Exemplo para dar merge do branch *dev* para o *master*
```
git checkout master

git merge dev
```

## Pegar hash dos commits
```
git log
```
Para sair aperte *q*
